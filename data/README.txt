# download.py

This script is used to download datasets required for the project. It fetches data from predefined URLs and stores them in the appropriate directories under `/data`. The name of the downloaded zip file needs to be changed to 'benchmark_imglist.zip'.
Script adopted from https://github.com/Jingkang50/OpenOOD/blob/main/scripts/download/download.py

## Usage

Run the script using:

```bash
python download.py
